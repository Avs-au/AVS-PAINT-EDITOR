# AVS Paint Launcher

**AVS Paint Launcher** is a small and practical project that allows you to quickly open **AVS Paint (Unofficial)** on Windows using a `.bat` file.

This launcher includes an interactive menu that allows you to:

- Open AVS Paint with one click.

- Easily exit from the console.

- Customize the executable path according to your installation.

This project is ideal for those who want quick access to AVS Paint or to share it with friends without complications.

## Requirements

- Windows
- AVS Paint installed

## Usage

1. Download `OpenAVS.bat`.

2. Edit the path within the file to point to where you have AVS Paint installed.

3. Double-click `OpenAVS.bat` to open Paint.
